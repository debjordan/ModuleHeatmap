## 🚀 O que é o ModuleHeatmap?

ModuleHeatmap é uma solução completa de **visualização de consumo de módulos** que monitora como seus usuários realmente utilizam cada parte do seu sistema. Diferente de ferramentas tradicionais de analytics, focamos especificamente no **engajamento por módulo** e **padrões de utilização** das funcionalidades da sua aplicação.
